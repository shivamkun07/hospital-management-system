# 🏥 Smart Hospital Management System

A desktop-based Hospital Management System developed using Python, SQLite, Tkinter, and CustomTkinter to streamline hospital operations and healthcare administration.

## 🚀 Features

### 👨‍⚕️ Patient Management
- Register patients
- View patient details
- Manage patient records

### 🩺 Doctor Management
- Add doctors
- Manage specializations
- Consultation fee management

### 👩‍⚕️ Nurse Management
- Add nurses
- Department and shift management

### 📅 Appointment Management
- Book appointments
- View appointments
- Cancel appointments

### 💊 Pharmacy Management
- Add medicines
- Track medicine stock
- Check medicine expiry dates

### 📋 Medical Records
- Store diagnosis details
- Manage prescriptions
- View patient history

### 🧪 Laboratory Reports
- Add lab reports
- View patient reports

### 💳 Billing System
- Generate bills
- Automatic total calculation

### 🔍 Additional Features
- Search records
- Update records
- Delete records
- Dashboard statistics
- SQLite database integration

## ▶️ How to Run

### 1. Clone the repository

```bash
git clone https://github.com/shivamkun07/hospital-management-system.git
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Run the application

```bash
python GUI.py
```
